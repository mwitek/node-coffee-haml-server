#### Get up and running:
```shell
npm install
```
(try sudo if you hit permission issues)
```shell
bower install
```
(you'll need bower for this command to work)


build distribution with:
```shell
gulp build
```

start server with:
```shell
gulp
```

start server without livereload:
```shell
gulp connect
```

default localhost url is http://localhost:5000
